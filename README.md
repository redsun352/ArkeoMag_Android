# ArkeoMag Pro - No Login Cleanroom

Bu proje eski APK'deki ana özellikleri temiz kodla yeniden kurar. Kullanıcı girişi, Firebase Auth, Play Integrity, PairIP lisans kontrolü ve reklam/analytics bağımlılıkları yoktur.

## Aktarılan/yeniden kurulan özellikler
- Dedektör ekranı
- Kalibrasyon ekranı
- Live Scan
- Area Scan / grid ısı haritası
- Sonar/3D relatif model ekranı
- Canlı X/Y/Z manyetometre ve |B| toplam değer
- Hard-iron offset kalibrasyonu
- Sesli alarm ve titreşim
- CSV dışa aktarım
- GPS koordinatıyla nokta kaydı
- Bluetooth eşleşmiş cihaz listeleme
- Eski projeden digital font, shader/model varlıkları ve layout referansları

## Build
GitHub'a push sonrası Actions sekmesinde APK artifact oluşur.
